1. clone된 디렉토리로 들어와 `npm install`
1. `node index.js` 실행 또는 Atom에서 `index.js`를 연 상태에서 `alt-R` (atom-runner 플러그인이 설치되어 있을 경우)
1. `getpost.html`을 웹브라우저에서 열어서 위에서 실행시킨 서버로 GET과 POST로 요청해보기
